
# Sauve qui peu — V3

Mini jeu runner web inspiré de Mario / Temple Run.

Personnage : MedhiMitchel
Bibliothécaire rebelle qui fuit la bibliothèque.

CONTROLES

PC
ESPACE ou flèche haut = sauter

SMARTPHONE
Toucher l'écran = sauter

FEATURES
- générateur d'obstacles
- score + highscore
- sprites pixel
- audio
- écran game over

DEPLOY

1 créer un repository GitHub
2 upload tous les fichiers
3 Settings → Pages
4 Deploy from main branch

Le jeu sera accessible sur
https://username.github.io/sauve-qui-peu
